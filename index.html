Selectors are the part of CSS rule set

CSS selectors select HTML elements according to its id, class, type, attribute etc.

There are several different types of selectors in CSS. CSS Element Selector.



pseudo selectors

Note: CSS pseudo-classes are used to add styles to selectors, but only when those selectors meet certain conditions.

Note: A pseudo class is expressed by adding a colon (:) after a selector in CSS, followed by a pseudo-class such as "hover", "focus", or "active", like this: a:hover { /* your style here */ }
